# rexwang.cloud

Rex Wang 个人品牌站 — 老板AI化方法论、开源产品、课程体系。

## 技术栈

- **Astro** — 静态优先框架，SEO 友好
- **Cloudflare Pages** — 免费托管，全球 CDN，国内可访问

## 站点结构

| 区域 | 内容 |

| 关于 | 个人简介 |

## 本地开发

```bash
npm install
npm run dev
```

访问 http://localhost:4321

## 部署
